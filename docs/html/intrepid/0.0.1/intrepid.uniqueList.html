Namespace: uniqueList

intrepid.uniqueList

UniqueList behaves basically like a cut-down array (only supports push at the moment), but only allows unique items. It does this using two methods. One for objects, and one for primive types. Objects are tracked by reference using a WeakMap, primitives are matched by value.
